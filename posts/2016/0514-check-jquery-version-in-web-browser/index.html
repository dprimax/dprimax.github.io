Check jQuery Version

jQuery is a JavaScript Library. jQuery greatly simplifies JavaScript programming. jQuery is easy to learn.

There is an easy way to check what is the jQuery version of the website. Run below short code inside Firebug or Chrome Developer Tools Console and shortly you will notice output like this:

> jQuery.fn.jquery
> "1.8.3"
